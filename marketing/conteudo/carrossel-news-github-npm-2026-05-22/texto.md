## Slide 02 — Body / como funciona
Eyebrow: `COMO O NOVO FLUXO FUNCIONA`
Subtítulo: **2FA + área de preparação antes da publicação.**

Corpo:
> O novo fluxo exigirá **autenticação de dois fatores** e, em vez de publicar diretamente com `npm publish`, desenvolvedores podem enviar módulos para uma área de preparação com `npm stage publish`.
>
> A mudança busca reduzir riscos em fluxos automatizados, que frequentemente usam tokens para autenticação, mas podem ser copiados e roubados.
