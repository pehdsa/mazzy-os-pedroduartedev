DRY | Don't Repeat Yourself
Abstraia padrões repetidos.
// ❌ Antes if (user.role === 'admin' || user.role === 'owner') { // ... } // em outro arquivo, a mesma checagem: if (user.role === 'admin' || user.role === 'owner') { // ... } // ✅ Depois const podeEditar = (user) => ['admin', 'owner'].includes(user.role); if (podeEditar(user)) { /* ... */ }

Concepts 06 / 07
05
Refatore sem medo
Sempre deixe melhor do que encontrou.
// ❌ Antes function calc(x, y) { if (x > 0 && y > 0) return x + y; else if (x < 0 && y < 0) return Math.abs(x) + Math.abs(y); return 0; } // ✅ Depois function somaAbsolutos(a, b) { const mesmoSinal = Math.sign(a) === Math.sign(b); return mesmoSinal ? Math.abs(a) + Math.abs(b) : 0; }
