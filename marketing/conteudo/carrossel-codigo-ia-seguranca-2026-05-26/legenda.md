# Legenda — IA acelera código e vulnerabilidades

IA acelera código. Vulnerabilidades também.

A saída de IA não vem revisada. Pode ter SQL injection, lib desatualizada, dados sensíveis em logs, permissões frágeis ou padrões inseguros copiados de exemplo qualquer.

O risco não está em usar IA pra programar. Está em tratar o output como se já tivesse passado pelo code review.

O dev não some. Muda. Vira validador.
